Search Savior is a Google Chrome Extension where a user will be able to use it to save the history of the tabs they go on the Chrome browswer during the period of time, the extension is being used. It has an On/Off Toogle to start/stop tracking. Once the user finishes browsing, the Search Savior will print out the Tree Structure on the webpage for the user to view it.
